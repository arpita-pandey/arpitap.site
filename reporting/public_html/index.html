CSE 135 Project

Analytics Reporting System

A role-based reporting platform for viewing analytics, managing reports, and sharing saved insights across different user access levels.

Included in this system

  • Role-based access control for Super Admin, Analyst, and Viewer accounts
  • Protected login flow with authenticated access to backend pages
  • Saved reports for read-only viewing by viewer accounts
  • Multiple report sections with charts, tables, and analyst commentary
  • Export support for report output and sharing
  • Structured dashboard experience for graders and test users
